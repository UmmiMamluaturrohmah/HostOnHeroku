Cara penggunaan website
1. Langkah pertama login ke localhost:8000/login-form. Atau tekan button "Log In".
2. Pilih metode Login sebagai guru atau siswa
3. Jika login sebagai guru, maka selanjutnya pilih kelas yang ingin dilihat data siswa yang terdapat pada eraport.
4. Setelah memilih data siswanya, selanjutnya klik button "Nilai" agar dapat melihat data nilai raport.
5. Dalam halaman Nilai tersebut dapat melakukan CRUD nilai raport.
6. Kemudian apabila website telah selesai digunakan maka Log Out.
7. Jika anda login sebagai siswa, maka anda hanya dapat melihat nilai raport yang anda dapatkan
8. Jika selesai melihat Nilai Raport anda dapat Log Out.
